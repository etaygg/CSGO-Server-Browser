# CS:GO Legacy Server Browser

An open-source desktop application designed to restore the ability to browse and connect to community servers in the **Legacy version of Counter-Strike: Global Offensive**.

After Valve transitioned from **CS:GO** to **CS2**, the built-in server browser stopped functioning properly for legacy builds. This project provides a lightweight alternative that allows players to discover and join community servers again.

## Features

**Live Server List**  
Fetches active community servers in real time using the **Steam Web API**.

**One-Click Connection**  
Directly connect to servers.

**Ping System**  
Measures latency (ping) for each server individually and prioritizes servers with the lowest ping by default.

---

## Getting Started

**Download**  
Download the latest release from the **Releases** tab and extract the ZIP file.

**Steam Web API Key**  
Create a personal API key from the [Steam Developer Portal](https://steamcommunity.com/dev/apikey) and paste it into the **Settings** section in the app.
